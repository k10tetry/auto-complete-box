## Installation

Add Jitpack to your project build.gralde file.

```javascript
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
```

Then add this dependency to your app build.gradle file.

```javascript
dependencies {
  ...
  implementation 'com.github.k10tetry:auto-complete-box:v1.0.0'
}
```
